Niagara Health Centre (NHS) Port Colborne Site Urgent Care Centre (UCC) MoHLTC Funding Submission

Architect                      McCallum Sather Architects
Partner-in-Charge                      Joanne McCallum
Project Architect                      Raminder Kanetkar
Year work completed                      2008
Project Value                      $1,000,000
Project Location                      Port Colborne, Ontario

Project Description:
Gross Floor Area: approx. 6,000sqft on 1 floor
This project involved re-purposing of the existing hospital spaces to an Urgent Care Centre. This was a stage 4 MoHLTC submission. The design involved exterior parking layout, construction of an accessible ramp, code review for occupancy, exiting and washrooms, interior space design, furniture allocation and construction of interior components such as new partitions, ceiling tiles, doors etc.

Unique Design Features:
The facility design needed many health, safety and fire protection related issues to be addressed.

Challenges Unique to the Project:
The project had to be completed on a tight budget and time schedule.

Project Involvement/Role:
As Project Architect, responsible for the interior and exterior design, Code analysis, MoHLTC submission including documents and reports for funding, and co-ordination with the consultants.

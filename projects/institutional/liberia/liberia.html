Liberia Library Proposal

Architect                      McCallum Sather Architects (MSA)
Partner-in-Charge                      Joanne McCallum
Project Architect                      Raminder Kanetkar
Year work completed                      2009
Project Value                      Unknown
Project Location                      Project Not Constructed

Project Description:
Gross Floor Area: approx. 10,900sqft in three (3) phases
Based upon her winning USGBC competition entry, Raminder Kanetkar spear headed this design for a library project in the war torn Country of Liberia. The presentation panels were put together to help raise funds in Canada for the building to be built in Liberia.

Project Involvement/Role:
Raminder Kanetkar as the Desin Team Leader was responsible for the design concept and the presentation panels.
